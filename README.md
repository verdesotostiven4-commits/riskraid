# RiskRaid

**RiskRaid** es un prototipo de juego de extracción para Roblox.

Idea principal:

> Entras a una raid, recoges loot, intentas extraer vivo y guardas lo conseguido en tu almacén. Si mueres, tu inventario cae en el mapa y otros jugadores pueden saquearlo.

## Estado actual

Base inicial del proyecto:

- Estructura compatible con Rojo.
- Mapa de prueba generado por servidor.
- Cajas de loot.
- Inventario de raid.
- Almacén temporal.
- Zona de extracción.
- Muerte con caída de mochila/loot bag.
- UI simple para probar.

## Cómo probarlo

1. Instalar Roblox Studio en PC o laptop.
2. Instalar Rojo o el plugin de Rojo para sincronizar este repo con Roblox Studio.
3. Abrir `default.project.json` con Rojo.
4. En Roblox Studio, presionar **Play**.

## Nota

Esta versión es solo el primer prototipo técnico. Todavía no tiene DataStore real, dúos, ranked, monetización, bots avanzados ni almacén 3D visitable.
