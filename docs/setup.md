# RiskRaid — Setup inicial

Esta guía es para abrir el proyecto en Roblox Studio usando GitHub + Rojo.

## Lo que necesitas

- Cuenta de Roblox: `SticlanPro`.
- Roblox Studio instalado en una PC o laptop.
- Git instalado.
- Rojo instalado o el plugin de Rojo configurado.
- Repositorio: `verdesotostiven4-commits/riskraid`.

## Flujo de trabajo

1. Yo actualizo el código en GitHub.
2. Tú descargas/sincronizas el repo en tu PC.
3. Abres Roblox Studio.
4. Con Rojo, sincronizas el proyecto.
5. Presionas **Play** y pruebas.
6. Me mandas errores/capturas.
7. Yo corrijo y actualizo GitHub.

## Comando típico con Rojo

En la carpeta del repo:

```bash
rojo serve default.project.json
```

Luego en Roblox Studio conectas el plugin de Rojo al servidor local.

## Prueba esperada

Al ejecutar el prototipo debe generarse:

- Arena gris.
- Paredes negras.
- Cajas azules de loot.
- Zona verde de extracción.
- Bloque rojo para probar muerte.
- UI de inventario y almacén.

## Importante

Nunca compartas tu contraseña de Roblox ni de GitHub. El flujo correcto es trabajar con el repositorio y que tú publiques desde tu propia cuenta.
